# Portifólio
Projeto do meu primeiro portifólio 
Trabalhando na evolução do meu portfólio online. Implementei dark mode, defini uma paleta própria e adicionei uma interação na foto de perfil com troca no hover. Foco em UI e experiência do usuário.

* HTML5
* CSS3
* JavaScript
* Bootstrap

## Objetivo

Este portfólio foi desenvolvido como parte do meu processo de aprendizado e evolução profissional, servindo como vitrine para oportunidades na área de tecnologia, com foco em desenvolvimento frontend e backend.

## Status do Projeto

Em desenvolvimento. Novas funcionalidades e melhorias serão adicionadas ao longo do tempo.
